Media Stall Shopping Cart


This is a project built together with my NUS-ISS team as a part of the coursework. The objective was to build a shopping cart web application using the Microsoft ASP.NET framework.

A user would be able to add items to a cart and the total price would be automatically calculated. A user was then able to remove items before checkout. If the user had previously logged in with a valid username and password, the cart would be saved even upon closing of the browser. Search functionality within the online store was also added.
